## Présentation
Nour_Gravity est une interface web immersive qui fusionne spiritualité, intelligence artificielle (Claude) et design antigravity. Elle permet de générer, structurer et visualiser des invocations personnalisées, alignées sur la puissance créatrice d’Al-Baqarah.

- **Matrice Invocation 99** : Génération dynamique d’attributs spirituels via API Claude
- **Résolution & Invocation Finale** : Formulation d’intentions/problèmes et génération d’une invocation curative exclusive
- **Effets visuels Antigravity** : UI premium, animations 3D, ambiance contemplative

## Stack technique
- Frontend : HTML5, Tailwind CSS, JS Vanilla (Antigravity, UI, logique)
- Backend : Node.js, Express, dotenv
- IA : Anthropic Claude 3.5 Sonnet (API)

## Installation & lancement
```bash
git clone https://github.com/El-hadj10/Nour_Gravity.git
cd Nour_Gravity
npm install
# Crée un fichier .env avec ta clé Anthropic
node server.js
```

Ouvre ensuite http://localhost:3000 dans ton navigateur.

## Fonctionnalités principales
- Génération d’invocations structurées (nom, arabe, translittération, essence)
- Mode fallback local si l’API est indisponible
- Résolution personnalisée : soumets un blocage, reçois une invocation adaptée
- UI responsive, animations antigravity, dark mode

## Sécurité
- La clé API doit rester dans `.env` (jamais committée)
- Aucune donnée sensible n’est stockée côté client
